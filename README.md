# Dictionary App 
​
 Aplicación sencilla de diccionario Inglés para Android.
​
 ## 🚀 Tecnologías
 *   **Jetpack Compose & Material 3** (UI)
 *   **MVVM** (Arquitectura)
 *   **Retrofit** (API REST)
 *   **Kotlin Coroutines** (Asincronía)

## 📖 API
 Datos obtenidos de [Free Dictionary API](https://dictionaryapi.dev/).
